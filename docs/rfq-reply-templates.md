# RFQ Reply Templates

## First reply

Thanks for your inquiry.

Please send the following details so we can check the quotation more accurately:

1. Insert code / drawing / sample photo
2. Workpiece material
3. Quantity
4. Required coating or grade, if any
5. Neutral packaging or private label packaging

For most carbide inserts, MOQ starts from 10 pcs. Standard carbide end mills can start from 1 pc.

## Model list reply

Thanks for your model list.

We will sort the items by product family first, including turning inserts, milling inserts, threading inserts, grooving inserts, PCD, CBN, cermet and carbide end mills. After checking availability, we will confirm MOQ, packing and quotation details.

## Private label reply

Yes, private label packaging can be supported for repeat and OEM orders.

Please send:
1. Label design or reference photo
2. Inner box quantity
3. Model label format
4. Annual or repeat order plan
5. Required packing language

## PCD reply

For PCD inserts, please send the drawing or sample photo first. Material, surface finish requirement and quantity are also important for quotation.

## CBN reply

For CBN inserts, please send material hardness, cutting condition, current insert code or sample photo, and quantity.
